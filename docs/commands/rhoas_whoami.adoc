= rhoas whoami

[role="_abstract"]
Output the current username

[discrete]
== Synopsis

View the username of the current user.

This command outputs the username for the user currently logged in.


....
rhoas whoami [flags]
....

[discrete]
== Examples

....
# Output the current username
$ rhoas whoami

....
