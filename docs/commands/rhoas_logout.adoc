= rhoas logout

[role="_abstract"]
Log out from RHOAS

[discrete]
== Synopsis

Log out from RHOAS. To manage your application services again, log in using ”rhoas login”.

....
rhoas logout [flags]
....

[discrete]
== Examples

....
rhoas logout

....
