== rhoas

rhoas cli

=== Synopsis

Work with your Managed Services

=== Examples

....
$ rhoas login
$ rhoas serviceaccount list
$ rhoas kafka create
....

=== Options

....
  -d, --debug   Enable debug mode
  -h, --help    help for rhoas
....
