== rhoas kafka topic list

List Kafka topics

=== Synopsis

List all topics in a Kafka instance

....
rhoas kafka topic list [flags]
....

=== Examples

....
# list all topics
$ rhoas kafka topic list

# list all topics as JSON
$ rhoas kafka topic list -o json
....
