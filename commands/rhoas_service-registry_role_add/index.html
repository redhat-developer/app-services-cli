rhoas_service-registry_role_add

rhoas service-registry role add#

Add or update principal role

Synopsis#

Add or update role for user or service account

rhoas service-registry role add [flags]

Examples#

## Create or update user rolerhoas service-registry role add --role=ADMIN --username=joedough

Options#

      --instance-id string       ID of the Service Registry instance to be used. By default, uses the currently selected instance      --role string              Role to apply      --service-account string   ServiceAccount name      --username string          Username of the user within organization
